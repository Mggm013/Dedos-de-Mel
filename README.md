# 🍬 Honeydukes — Dedos de Mel

Site fictício da loja de doces **Honeydukes**, inspirada no universo de Harry Potter. Desenvolvido como projeto de aprendizado de HTML, CSS e Bootstrap.

---

## 📋 Sobre o Projeto

A Honeydukes é uma loja de doces mágicos localizada em Hogsmeade. Este site foi desenvolvido para simular uma loja virtual completa, com navegação, produtos, formulários e muito mais.

---

## 🚀 Funcionalidades

- ✅ Navbar responsiva com dropdown de produtos
- ✅ Carrossel de imagens na página inicial
- ✅ Cards de produtos com carrinho funcional
- ✅ Sistema de favoritos nos produtos
- ✅ Modal de Login e Cadastro
- ✅ Seção de depoimentos de clientes
- ✅ Página Sobre Nós com equipe
- ✅ Página de Contato com formulário e mapa
- ✅ Página Trabalhe Conosco com accordions
- ✅ Rodapé com links e redes sociais
- ✅ Layout totalmente responsivo

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- [Bootstrap 5.3.8](https://getbootstrap.com/)
- [Bootstrap Icons 1.11.3](https://icons.getbootstrap.com/)
- JavaScript (vanilla)
